# BarangBareng

Marketplace sewa dan pinjam barang mahasiswa.

Satu-satunya versi SPA aktif berada di:

```text
public/spa/
```

Jalankan lokal lewat Next:

```bat
npm run dev
```

Lalu buka:

```text
http://localhost:3000
```

Root Next langsung mengarahkan browser ke `public/spa/index.html`, jadi tampilan lokal dan deploy memakai halaman yang sama tanpa wrapper iframe.
